# ANKOR Voice Agent

Real-time voice agent for ANKOR evaluations using Google ADK Streaming and the Gemini Live API.

## Requirements

- Python 3.11+
- Google Cloud project with Vertex AI enabled
- Supabase Auth access tokens from the ANKOR client

## Environment

Create a local env file using `.env.example` as reference:

- `APP_NAME`: App name passed to ADK Runner
- `LIVE_MODEL`: Live API model (default `gemini-live-2.5-flash-native-audio`)
- `GOOGLE_GENAI_USE_VERTEXAI`: set `1` for Vertex AI
- `GOOGLE_API_KEY`: AI Studio API key (when `GOOGLE_GENAI_USE_VERTEXAI=0`)
- `GOOGLE_CLOUD_PROJECT`: GCP project id
- `GOOGLE_CLOUD_LOCATION`: e.g. `us-central1`
- `ANKOR_API_BASE_URL`: Supabase Functions API base URL
- `HTTP_TIMEOUT_S`: HTTP timeout (seconds)
- `MANUAL_ACTIVITY_SIGNALS`: set `1` to send manual activity start/end
- `EXPOSE_TOOL_EVENTS`: set `1` to stream tool calls/results to the client

## Install

```bash
pip install -e .
```

## Run

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

## WebSocket Protocol (Summary)

1. Connect to `ws://<host>:8000/ws/voice`
2. Send `{ "type": "auth", "access_token": "<supabase_access_token>" }`
3. Send `{ "type": "session_init", "org_id": "...", "team_id": "...", "coach_id": "...", "evaluation_flow": "create_evaluation", "locale": "en-US" }`
4. Stream audio chunks `{ "type": "input_audio_chunk", "data": "<base64_pcm16le>", "sample_rate_hz": 16000, "channels": 1, "format": "pcm16le" }`

## Notes

- Tokens are generated by Supabase Auth in the client and forwarded to this service.
- This service forwards the token to the Supabase Functions API via `Authorization: Bearer <token>`.
- Session storage is in-memory only (use a persistent session service for production scale).

## AI Studio vs Vertex AI

- **AI Studio (dev/prototype)**:
  - Set `GOOGLE_GENAI_USE_VERTEXAI=0`
  - Set `GOOGLE_API_KEY` in your env
- **Vertex AI (production)**:
  - Set `GOOGLE_GENAI_USE_VERTEXAI=1`
  - Set `GOOGLE_CLOUD_PROJECT` and `GOOGLE_CLOUD_LOCATION`
  - Use `gcloud auth application-default login` to provide ADC credentials
